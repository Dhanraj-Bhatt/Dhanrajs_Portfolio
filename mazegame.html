PC JOYSTICK GAME

Maze Game

Maze Game is an exciting joystick-based game where players can control a ball moving the joystick. The objective is to navigate through a maze and reach the end. The game features 5 different levels, with each level becoming more difficult than the last. Players can choose which level they want to play, allowing them to start easy and increase the challenge as they improve their skills.

This Project was created for Gayan Kunj Surat Event.

Benefits of the Project:

  • Fun Gameplay: The joystick control adds an interactive and enjoyable way to play the game.
  • Stress Relief: Playing a fun game like this can help players relax and unwind after a long day.
