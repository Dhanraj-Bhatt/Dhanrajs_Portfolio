VR Training With Sensors And IoT

Fire Safty Training

Fire Safty Training is a VR training application designed to teach users how to handle fire hazard situations in a safe and immersive way. The application is built in Unity using the PICO XR SDK to create a realistic 3D training environment.

The project uses motion sensors to track a real physical fire extinguisher inside VR, while a custom IoT device communicates the extinguisher trigger press. It also includes a task manager that checks whether the user follows the correct procedure, including selecting the right fire extinguisher type and completing the required training steps properly.

Training Highlights

  • Practical Safety Training: Users can learn how to respond to fire hazards through an interactive real-time simulation.
  • Real-World Interaction: Motion sensors and IoT integration connect physical extinguisher actions with the virtual environment.
  • Procedure Validation: The task manager verifies whether the user completes the training correctly step by step.
  • Correct Equipment Awareness: Helps trainees understand how to choose the proper fire extinguisher type for the situation.
  • Safe Practice Environment: Allows users to build confidence without facing real fire risk during training.
