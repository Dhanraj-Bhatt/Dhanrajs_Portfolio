Amaron Battery Showcase and Quiz Game

The Amaron Battery Showcase and Quiz Game project was designed to highlight the variety of batteries offered by Amaron for different machines and applications. The application features an interactive product showcase, making it easy for users to explore and learn about the batteries. It also includes a fun and engaging quiz game related to the products, helping users test their knowledge in an enjoyable way.
this project was made for Automechanika Dubai Exhibition.
