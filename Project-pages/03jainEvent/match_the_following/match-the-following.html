Lakshy Link Up

Lakshy Link Up is a fun and interactive game where players match images from the left side to their related images on the right side by dragging a wire string between them. The game includes 10 images on each side, you have 45 sec to complete this game, shuffled for every new game, ensuring a fresh experience each time you play. It's a great way to challenge your memory and improve your matching skills!

Benefits of the Project

  • Enhances visual memory and concentration.
  • Fun and relaxing gameplay for all ages.
  • Every round feels new and exciting with shuffled images.

This Project was created for Gayan Kunj Surat Event.
