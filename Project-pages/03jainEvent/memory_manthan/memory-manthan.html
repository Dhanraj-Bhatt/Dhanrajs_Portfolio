Memory Manthan

Memory Manthan is an exciting memorizing game designed to test and improve memory skills. In the game, the user is shown 10 different images for 10 seconds. After this, the images are hidden, and the player has to memorize them. In the next scene, the user must select the 10 images from a set of 50 different images within 20 seconds. The challenge is to remember the images accurately and quickly!

Fun Benefits of the Project:

  • Improves Memory: This game helps players improve their short-term memory and recall skills.
  • Enhances Focus: It encourages players to stay focused while remembering the images and making quick decisions.
  • Challenging and Fun: The time limits make the game exciting, and players can challenge themselves to beat their best score.
